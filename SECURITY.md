# Security Policy

## Supported Versions

Security fixes are applied to the latest released version only.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, email **andy327@gmail.com** with the subject line `[sparkling] Security Vulnerability`. Include as much detail as possible:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a minimal proof of concept
- Any relevant Scala, Spark, or JDK version information

You can expect an acknowledgement within 48 hours and a resolution or status update within 7 days.
